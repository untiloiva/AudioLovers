# AudioLovers
- Sovelluksessa käyttäjät pystyvät jakamaan musiikkia .mp3 tiedostoina ja jättää kommentteja sekä arvostelun muiden kappaleisiin.
- Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
- Käyttäjä pystyy lisäämään ja poistamaan musiikkia sovelluksesta.
- Käyttäjä pystyy kommentoimaan muiden musiikkikappaleisiin ja antamaan niille arvosanan asteikolla 0/5.
- Käyttäjä näkee sovellukseen lisätyn musiikin, sen arvosanan ja muiden kommentit sekä arvosanat.
- Käyttäjä pystyy etsimään musiikkia hakusanalla.
- Käyttäjäsivu näyttää kappaleet, jotka käyttäjä on lisännyt sekä kommentit ja arvosanat, jotka käyttäjä on jättänyt.
- Käyttäjä pystyy valitsemaan musiikilleen yhden tai useamman luokittelun eri musiikin genreistä (esim. rap, pop, rock, experimental)
